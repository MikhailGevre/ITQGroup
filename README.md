## 🚀 Project Setup

Запуск приложения:

Перед запуском приложения, выдайте права на скрипт по созданию файлов .env, properties

```bash
   chmod +x scripts/init.sh
   chmod +x scripts/create-props.sh
   ```
Запуск самого скриптов. После создания скрипта, проиницилизируйте поля.
```bash
   scripts/init.sh
   ./scripts/create-properties.sh
```

API по поиску документов производится по созданию документа
